## AI 도용 이미지 탐지 시스템 개발

```
[작가의 원본 이미지]
        │
        ▼
[CLIP 임베딩 추출]
        │
        ▼
[유사 이미지 수집 (웹 크롤링 / 검색)]
        │
        ▼
[CLIP 임베딩 추출 (유사 이미지들)]
        │
        ▼
[임베딩 간 유사도 계산 (cosine)]
        │
        ▼
[유사도 임계값 기반 필터링 + 리포트 생성]
```

### CLIP 란?

이미지와 텍스트를 같은 임베딩 공간으로 매핑하여 **"이 이미지가 이 문장과 얼마나 관련 있는가"**를 비교
